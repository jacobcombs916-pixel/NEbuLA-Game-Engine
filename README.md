# NEbuLA-Game-Engine
NGE Is an open-source game engine and written in HTML, and easy to remember. You can use ANY programming language you want, like... Common Programming Language File Extensions C and C++ C Source Files: .c, C Header Files: .h C++ Source Files: .cpp (most common), .cc, .cxx C++ Header Files: .hpp, .hxx, .h (legacy C++ headers). And a text editor, also a "SAVE TO FOLDER", very fancy and cool animations, even a custom icon file!
